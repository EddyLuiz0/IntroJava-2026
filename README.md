Este repositório foi criado com o objetivo de armazenar e acompanhar as atividades práticas propostas pelo professor durante as aulas da disciplina de Programação em Java no curso de Análise e Desenvolvimento de Sistemas.

Aqui estão reunidos exercícios, desafios e pequenos projetos desenvolvidos ao longo do aprendizado, com foco na prática da lógica de programação e no domínio dos principais conceitos da linguagem Java.

🎯 Objetivos do repositório

Praticar lógica de programação

Aplicar conceitos fundamentais da linguagem Java

Desenvolver soluções para problemas propostos em aula

Registrar minha evolução no aprendizado de programação

🧠 Conteúdos praticados

Entre os principais conceitos trabalhados estão:

Estruturas condicionais (if, else, switch)

Entrada de dados com Scanner

Estruturas de repetição

Variáveis e tipos de dados

Operadores lógicos e matemáticos

Desenvolvimento de pequenos sistemas em Java

💻 Sobre os exercícios

Os códigos presentes neste repositório representam atividades práticas realizadas durante os estudos, como por exemplo:

Sistemas simples de pedidos

Cálculo de valores e descontos

Exercícios de lógica

Simulações de problemas do dia a dia utilizando programação

🚀 Objetivo pessoal

Este repositório também funciona como um registro da minha evolução na área de tecnologia, onde estou constantemente praticando, aprendendo novos conceitos e desenvolvendo minhas habilidades em programação.
